# WV Website

```
git clone https://github.com/willemsventures/voyager-website
cd website
yarn install
node voyager.js
```

for live editing use "pm2 start voyager.js --watch" instead of "node voyager.js".

```
pm2 start voyager.js --watch
```

Website visible at "https://localhost:8081"

Under development.
